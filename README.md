# nono-card-unlock

## Purpose
Automate KONAMI's nono card game, a luck-based game where you earn stamps that can be used to unlock songs based on the card selected. Play the game [here](https://p.eagate.573.jp/game/bemani/wbr2020/01/card.html) (requires an account, game ends at 10/29 23:59 JST).

## Usage
Run the script using `python main.py`. You can enable debug logging by adding the `--debug` argument.  
Required dependencies: requests, scikit-image, opencv, beautifulsoup4  
Tested and works on Python 3.8.3.
